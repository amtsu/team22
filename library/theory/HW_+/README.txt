================================================================================================

Для работы тестов в файле дополнительного домашнего задания необходимо установить модуль ipytest

================================================================================================

1. Откройте терминал на своем сервере absrent.ru

2. Введите в терминале команду:

    pip install ipytest
    
3. Дальше произойдет много магии

4. Когда все загрузится и вы снова сможете вводить команду в терминале, можете закрывать терминал
и идти делать домашнее задание с проверкой в виде тестов

    Желаю успехов!!!
